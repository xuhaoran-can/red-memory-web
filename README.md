# 🚀 红色记忆——济南英雄风云录 | 全栈升级版

## ✨ 新功能亮点

### 1️⃣ **登录注册系统** 🔐
- 精美的登录/注册页面（保持红色文化主题）
- JWT令牌认证（24小时有效期）
- 密码加密存储（bcrypt）
- 自动登录状态保持

### 2️⃣ **SQLite真实数据库** 🗄️
- **9张数据表**：
  - `users` - 用户信息
  - `medals` - 勋章定义
  - `user_medals` - 用户勋章关联
  - `score_history` - 积分变动记录
  - `questions` - 题目库
  - `quiz_attempts` - 答题记录
  - `clues` - 线索库
  - `clue_purchases` - 购买记录
  - `activity_stats` - 活跃度统计

### 3️⃣ **真实排行榜** 🏆
- **从数据库动态查询**前10名用户
- **实时更新排名**（每次答题后刷新）
- **三种排行模式**：
  - 总积分排行
  - 今日排行
  - 本周排行
- **我的排名**显示（即使不在前10名）

## 🔌 API接口文档

### 用户认证
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/auth/register` | 用户注册 |
| POST | `/api/auth/login` | 用户登录 |
| GET | `/api/auth/me` | 获取当前用户 |

### 积分系统
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/user/score` | 获取积分 |
| POST | `/api/user/score/add` | 增加积分 |
| POST | `/api/user/score/deduct` | 扣除积分 |

### 游戏数据
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/medals` | 获取勋章列表 |
| POST | `/api/medals/:id/unlock` | 解锁勋章 |
| GET | `/api/questions` | 获取题目列表 |
| POST | `/api/questions/:id/answer` | 提交答案 |
| GET | `/api/clues` | 获取线索列表 |
| POST | `/api/clues/purchase` | 购买线索 |

### 排行榜（⭐核心功能）
| 方法 | 路径 | 参数 | 说明 |
|------|------|------|------|
| GET | `/api/leaderboard` | type=score\|today\|week, limit=10 | **真实排行榜查询** |

## 💾 数据持久化说明

### 与旧版本的区别

**❌ 旧版（localStorage）：**
- 数据存储在浏览器本地
- 清除浏览器数据会丢失所有进度
- 排行榜是模拟数据（假数据）
- 无法多设备同步

**✅ 新版（SQLite数据库）：**
- 数据保存在服务器 `database/red_memory.db`
- 永久保存，不会丢失
- **排行榜是真实用户数据**
- 支持多用户同时在线
- 所有操作都有记录可查

### 数据库文件位置
```
f:\桌面\大二作业\自己做的作品\ai对话网页\database\red_memory.db
```

## ❓ 常见问题

### Q: 启动服务器报错 "Cannot find module 'xxx'"？
A: 运行 `npm install` 安装所有依赖。

### Q: 访问页面显示"未登录"？
A: 先访问 `http://localhost:3000/login.html` 登录。

### Q: 排行榜没有数据？
A: 需要先注册账号并答题，排行榜才会显示真实数据。

### Q: 如何添加更多题目？
A: 编辑 `database/init.js` 中的 `questionsData` 数组，然后重新初始化数据库。

### Q: 可以部署到线上吗？
A: 可以！推荐平台：
- Vercel / Netlify (免费)
- 阿里云 / 腾讯云
- 自己的VPS服务器
